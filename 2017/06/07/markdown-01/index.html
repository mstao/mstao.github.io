列表

Markdown 支持有序列表和无序列表。

无序列表使用星号、加号或是减号作为列表标记：

*   Red
*   Green
*   Blue
等同于： 
+   Red
+   Green
+   Blue
也等同于： 
-   Red
-   Green
-   Blue
有序列表则使用数字接着一个英文句点： 
1.  Bird
2.  McHale
3.  Parish
